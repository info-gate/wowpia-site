## Why HWP Files Won't Open on iPhone by Default

HWP is a proprietary format developed by Hancom, the South Korean software company behind the Hangul Word Processor suite. It's been the standard document format for Korean government agencies, schools, and businesses for decades.

The problem for iPhone users is straightforward:

- Apple's **Files** app doesn't support HWP
- **Microsoft Word** on iOS cannot open HWP files
- **Google Docs** also lacks native HWP support
- Even desktop Mac users need a third-party solution

This leaves a lot of people — especially Korean expats, international students, and anyone dealing with Korean administrative paperwork — stranded every time an `.hwp` file lands in their inbox.

The good news: the workarounds are simple, and the best ones are completely free.

## Method 2: Convert HWP to PDF Online Before Opening

If you only need to view an HWP file occasionally and don't want to install an extra app, converting the file to PDF first is a reasonable option.

### Steps for online conversion

1. On your iPhone, open Safari and search for an HWP-to-PDF converter (several free options exist)
2. Upload your `.hwp` file from your Files app
3. Download the converted PDF
4. Open it with any PDF viewer — including Apple's built-in **Books** app or **Files** preview

### Limitations to keep in mind

| Factor | Online Converter | Dedicated App (OpenDocs) |

| File size limits | Often 5–20 MB | None |
| Privacy | File uploaded to server | Stays on your device |

For sensitive documents — tax forms, medical records, legal paperwork — uploading to a third-party server is worth thinking twice about. A local app keeps everything on your device.

## Method 3: Open HWP Files via Cloud Storage (Google Drive Workaround)

Google Drive added limited HWP viewing support in recent years. If someone shares an HWP file with you via Google Drive, you may be able to preview it directly in the Drive app without downloading anything.

### How to try the Google Drive method

1. Open the **Google Drive** iOS app
2. Navigate to the shared HWP file
3. Tap the file to open the preview
4. If Drive supports the file, it renders in-app — otherwise, you'll see a "Can't preview" error

### Why this isn't always reliable

Google Drive's HWP support is inconsistent. Complex documents — those with tables, images, headers, or special Korean fonts — often fail to render correctly or don't open at all. This method works best for simple, text-heavy HWP files.

For anything more complex, fall back to Method 1.

## What to Do When an HWP File Still Won't Open

If you've tried the steps above and the file still isn't opening correctly, here are the most common causes and fixes:

**The file is corrupted or incomplete**
→ Ask the sender to re-send the file. HWP files can get corrupted during some messaging apps' compression.

**The file was sent as a link, not an attachment**
→ Some Korean portals (like government websites) require a logged-in session to download. Try opening the download link in Safari on iPhone, then use the share sheet to pass it to OpenDocs.

**The file is an older HWP format (HWP 2.x or 3.x)**
→ Very old Hangul documents use legacy formats that even modern viewers sometimes struggle with. Try converting to PDF first using Method 2.

**The app didn't register as a share target**
→ After installing OpenDocs, try restarting your iPhone. iOS sometimes needs a reboot to refresh the list of apps available in the share sheet.
