本日、Barolingo のランキングに初めて実ユーザーの方々が登録されました。お名前はわかりませんが、画面越しにご挨拶を — 本当にありがとうございます。

私たちは小さなインディーアプリスタジオです。毎日お一人ずつ増えていくことに大きな意味があります。いただいたご意見を素早く反映し、1.0.4 アップデートでお返しします。

1.0.4 — 何が変わったか

1. 通知タップ・再起動でもすぐに学習画面へ

学習通知をタップしても、アプリを再起動してもオンボーディング画面が再表示されるバグがありました。ログイン状態が維持され、メイン画面に直接入るように修正しました。

技術的には、トークンによるセッション復元 → /me 取得 → 正確な分岐（メイン / オンボーディング / サインイン）の流れに再設計しました。

2. 💡 チャンク「詳しく」解釈 — 10言語自動対応

英語チャンクの短い翻訳だけでは物足りない時がありますよね。辞書画面で各チャンク横の 「💡 詳しく」ボタン をタップすると、AI が 4層解釈 を表示します：

  • 直訳 — 単語一つずつの意味
  • 自然な翻訳 — ネイティブが実際に使う自然な表現
  • ニュアンス — どんな場面で使うか（フォーマル/カジュアル）
  • 似た表現 — 同じ意味の別の英語表現

韓国語ユーザーは韓国語で、英語学習者が日本人なら日本語で、トルコ・モンゴル・ベンガル・ネパール・ミャンマー・カザフ・ウズベクのユーザーも 自動で母国語で 解釈を受けられます（合計10言語）。

技術的には、初回呼び出し時のみ LLM (Claude Haiku) が解釈を生成しデータベースにキャッシュ保存 → 次回からは即時表示。他のユーザーが同じチャンクを同じ言語で見ると、その方はキャッシュされたものを受け取ります。

3. お問い合わせメール変更

お問い合わせメールを wowpia0127@gmail.com に統一しました（設定 + ヘルプ画面）。
