아이폰에서 HWP 파일 열기 — 3가지 방법 완전 정리

아이폰에서 HWP 파일 열기가 안 된다면? 기본 앱부터 무료 전용 뷰어까지, 2026년 최신 방법 3가지를 한눈에 정리했습니다. 바로 해결하세요.

아이폰에서 HWP 파일을 열려면 전용 뷰어 앱이 필요합니다. iOS 기본 앱(파일, 메일 등)은 HWP 형식을 지원하지 않기 때문입니다. 아래 방법 중 하나만 선택하면 수십 초 안에 해결됩니다.

왜 아이폰은 HWP 파일을 바로 못 열까?

HWP(한글과컴퓨터 전용 포맷)는 국내 공공기관·학교·기업에서 표준처럼 쓰이지만, 애플의 기본 앱은 이 형식을 공식 지원하지 않습니다.

  • 파일 앱: PDF·DOCX·이미지는 열리지만 HWP는 미지원
  • 메일·메시지: 첨부 파일 미리보기 불가
  • Pages: HWP 가져오기 기능 없음

결국 "지원되는 앱이 없습니다"라는 오류 메시지만 뜨게 됩니다.

방법 1 — 무료 HWP 뷰어 앱 사용하기 (가장 빠른 해결책)

전용 앱을 설치하면 변환 과정 없이 HWP 파일을 바로 열 수 있습니다.

OpenDocs는 HWP를 포함해 PDF, DOCX, XLSX까지 한 앱에서 무료로 지원합니다.

사용 방법

  1. App Store에서 OpenDocs 설치
  2. 메일·카카오톡·파일 앱에서 HWP 파일 터치
  3. 공유 시트 → "OpenDocs로 열기" 선택
  4. 즉시 문서 내용 확인 가능

표, 이미지, 글꼴이 포함된 복잡한 HWP도 레이아웃 깨짐 없이 표시됩니다.

방법 2 — HWP를 PDF로 변환 후 열기

앱 설치가 어렵거나 단순히 내용 확인만 필요할 때 유용합니다.

PC에서 변환 후 아이폰으로 전송

  1. PC에서 한글(HWP) 프로그램 열기
  2. 파일 → 다른 이름으로 저장 → PDF 선택
  3. 변환된 PDF를 iCloud Drive 또는 카카오톡으로 전송
  4. 아이폰 파일 앱 또는 기본 뷰어로 열기

온라인 변환 사이트 활용

인터넷 브라우저에서 HWP-to-PDF 변환 서비스를 이용할 수 있습니다.
단, 개인정보가 담긴 문서는 외부 서버 업로드에 주의하세요.

방법 3 — 한컴독스(한글과컴퓨터 공식 앱) 사용하기

한컴에서 직접 만든 모바일 앱입니다. HWP 편집까지 지원하지만 일부 기능은 유료입니다.

기능 한컴독스 OpenDocs
HWP 보기 ✅ ✅
HWP 편집 ✅ (유료) ❌
PDF·DOCX 보기 ✅ ✅
완전 무료 ❌ ✅
용량 다소 큼 가벼움

편집이 필요하면 한컴독스, 보기만 할 거라면 무료 뷰어가 효율적입니다.

카카오톡·이메일에서 HWP 파일 받았을 때

일상에서 가장 흔한 상황입니다.

카카오톡 첨부 파일:

  1. 파일 탭에서 HWP 터치
  2. 우측 상단 공유(↑) 아이콘 탭
  3. "OpenDocs로 열기" 선택

이메일 첨부 파일:

  1. 첨부 파일 길게 누르기
  2. 공유 → OpenDocs 선택

앱을 한 번만 설치해 두면 이후엔 두 탭만으로 해결됩니다.

자주 묻는 질문

Q. 아이폰에서 HWP 파일을 편집할 수 있나요?
뷰어 앱 대부분은 보기 전용입니다. 편집이 필요하면 한컴독스 유료 플랜이나 PC 환경을 이용하는 것이 현실적입니다.

Q. HWP 파일이 깨져 보여요.
특수 글꼴이나 구버전 HWP 포맷일 때 발생합니다. 다른 뷰어 앱을 시도하거나, PC에서 PDF로 변환 후 열어보세요.

Q. 아이패드에서도 같은 방법이 통하나요?
네. iOS와 iPadOS는 동일한 앱 생태계를 공유하므로 같은 방법으로 해결됩니다.

결론 — 가장 빠른 선택은?

상황 추천 방법
지금 당장 열어야 한다 OpenDocs 설치
PC가 있고 시간 여유 있다 PDF 변환 후 전송
편집까지 필요하다 한컴독스 유료 플랜

아이폰에서 HWP 파일 열기로 고생하고 있다면 OpenDocs를 먼저 시도해 보세요. PDF, DOCX, XLSX도 함께 지원하니 별도 앱을 여러 개 깔 필요가 없습니다. 무료로 설치할 수 있어 부담도 없습니다.
