첫 학습자분들께

오늘 처음으로 Barolingo 랭킹에 실제 사용자 분들이 등록되셨습니다. 이름은 모르지만 화면 너머로 인사드립니다 — 고맙습니다.

작은 인디 앱 스튜디오에서 만들고 있는 앱이라 매일 한 분씩 늘어나는 게 큰 의미입니다. 받은 의견 빠르게 반영해서 1.0.4 업데이트로 돌려드립니다.

1.0.4 — 무엇이 바뀌었나

1. 알람·재실행 시 바로 학습 화면으로

이전엔 학습 알람을 눌러도, 앱을 다시 켜도 온보딩 화면이 또 나오는 버그가 있었습니다. 로그인 상태가 유지되어 바로 메인 화면으로 들어가도록 고쳤습니다.

기술적으로는 토큰 복원 → 사용자 정보 자동 fetch → 정확한 분기 (메인 / 온보딩 / 로그인) 흐름으로 재설계.

2. 💡 청크 "자세히" 해석 — 10개 언어 자동 지원

영어 청크 옆에 짧은 한글 힌트만 있으면 가끔 부족하실 때가 있죠. 이제 사전 화면에서 각 청크 옆 "💡 자세히" 버튼을 누르면 AI 가 4단계 해석을 보여드립니다:

  • 직역 — 단어 하나하나의 뜻
  • 자연스러운 번역 — 원어민이 실제로 쓰는 자연스러운 표현
  • 뉘앙스 — 언제 어떤 상황에서 쓰는지 (격식/캐주얼)
  • 비슷한 표현 — 같은 의미의 다른 영어 표현들

한국어 사용자는 한국어로, 영어 학습자가 일본인이면 일본어로, 터키·몽골·벵갈·네팔·미얀마·카자흐·우즈벡 사용자도 자동으로 자기 모국어로 해석을 받습니다 (총 10개 언어).

기술적으로는 첫 호출 시만 LLM (Claude Haiku) 이 해석을 생성하고 데이터베이스에 캐시 저장 → 다음부터는 즉시 표시. 다른 사용자가 같은 청크를 같은 언어로 보면 그분은 캐시된 것을 받습니다.
